# scale_rows

Normalize (z-score) rows of a matrix

## Usage

``` r
scale_rows(x)
```

## Arguments

- x:

  a matrix, possibly containing gene by samples

## Value

Row-normalized matrix

## Examples

``` r
# \donttest{
library(cinaR)
data(atac_seq_consensus_bm) # calls 'bed'
bed.row.normalized <- scale_rows(bed[,c(4:25)])
```
